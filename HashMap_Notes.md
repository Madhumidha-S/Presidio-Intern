# HashMaps in Python

- Creating dict - {}, dict()
- Accessing dict - dict[key]
- Updating dict(new or existing) - dict[key] = value
- Delete or Pop - del dict[key], x = dict.pop('key')

#### Advantages 

- Fast lookups - O(1) avg complexity
- Flexibility - support diverse types of keys
- Dynamic size
- Rich Functionality
- Easy-to-Read Syntax

#### HashMap Operations 

- Checking the Existence of Keys
- Looping
- Merging two HashMaps

#### When to Avoid HashMaps

- When Order Matters
- For Sequential Access
- Memory Constraint
- For Small, Fixed Collections

#### Advanced Techniques

- Default Dict
- Nested Dict
- Dictionary Comprehension
